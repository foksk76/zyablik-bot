# Identity Plugin Documentation

MAX Identity Bot plugin для получения `user_id` / `chat_id` из МАХ.

## Overview

Bot-platform identity plugin обрабатывает входящие события МАХ и возвращает информацию о получателе для настройки Zabbix.
