# Проект: Бот в МАХ для Zabbix-уведомлений

В рамках проекта планируется реализовать интеграцию Zabbix с ботом в МАХ для автоматической рассылки уведомлений о событиях мониторинга, требующих внимания заинтересованных получателей.

**Цель проекта** — организовать доставку оперативных уведомлений из Zabbix в корпоративный мессенджер МАХ для ответственных и заинтересованных в получении сотрудников.

На первом этапе проект ограничивается задачей доставки уведомлений из Zabbix. Бот используется как транспортный канал для оперативного информирования получателей о событиях, инцидентах, сбоях и восстановлении сервисов, контролируемых в Zabbix.

## Входит в этап

- создание и настройка бота;
- настройка механизма отправки уведомлений из Zabbix в МАХ;
- определение формата уведомлений в зависимости от необходимого уровня информативности для получателя;
- маршрутизация уведомлений по группам или получателям;
- проверка доставки тестовых событий из Zabbix;
- фиксация минимальной эксплуатационной документации.

## Ожидаемый результат

Повысить надежность доставки оповещений системы мониторинга ответственным за сервисы до целевого уровня 100% за счет дублирования уведомлений в двух независимых каналах — Telegram и МАХ.
