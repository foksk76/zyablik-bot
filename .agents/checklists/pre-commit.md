# Pre-commit checklist

Перед завершением задачи проверь:

- [ ] Изменения соответствуют текущей границе проекта.
- [ ] README и документация не противоречат друг другу.
- [ ] Нет реальных токенов, паролей, внутренних URL и боевых идентификаторов.
- [ ] Нет конкретных организационных названий.
- [ ] Основной webhook-скрипт лежит в `src/zabbix-media-type/max-webhook.js`.
- [ ] Если изменено поведение webhook, обновлен `docs/zabbix-media-type.md`.
- [ ] Если изменена архитектура, добавлен ADR.
- [ ] Выполнен `bash scripts/verify-repo.sh` или проведена ручная проверка.
