# Контекст проекта для ИИ-агентов

Проект: Zabbix MAX Alert Bot.

Назначение: доставка уведомлений из Zabbix в корпоративный мессенджер МАХ для ответственных и заинтересованных получателей.

Граница текущего этапа: только транспорт уведомлений Zabbix -> МАХ. Без SIEM, без AI-аналитики, без автоматического реагирования.

Существующий рабочий канал Telegram не ломать и не заменять. МАХ добавляется как дублирующий канал для повышения надежности доставки.

В репозиторий не добавлять реальные токены, идентификаторы боевых чатов, внутренние URL Zabbix и чувствительные данные.

Основной артефакт: `src/zabbix-media-type/max-webhook.js`.

Стиль документации: по-русски, коротко, понятно для широкой аудитории инженерного ИТ-состава, без лишнего маркетинга и без привязки к конкретным организационным названиям.
