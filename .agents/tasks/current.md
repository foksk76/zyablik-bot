# Текущая задача

## Статус

Инициализация каркаса проекта для AI-assisted разработки.

## Цель

Сделать репозиторий удобным для работы с Codex и другими AI-агентами: добавить инструкции, процесс, чек-листы, шаблоны задач и базовую проверку.

## Граница

Не менять функциональность webhook без отдельной задачи.

## Definition of Done

- есть `AGENTS.md` в корне репозитория;
- есть рабочий контекст в `.agents/`;
- есть описание процесса разработки;
- есть шаблоны issue и pull request;
- есть простой проверочный скрипт;
- README ссылается на AI/Codex-каркас.
