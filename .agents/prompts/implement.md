# Prompt: реализация изменения

Используй этот шаблон для постановки задачи Codex.

```text
Задача: <кратко что нужно сделать>

Контекст:
- проект: Zabbix MAX Alert Bot;
- текущая граница: только доставка уведомлений Zabbix -> МАХ;
- не добавлять SIEM, AI-аналитику и автоматическое реагирование;
- не использовать реальные секреты и организационные названия.

Что изменить:
- <файл или область 1>
- <файл или область 2>

Критерии готовности:
- изменение минимальное и проверяемое;
- документация обновлена, если изменилось поведение;
- выполнен npm test;
- результат кратко описан.
```
